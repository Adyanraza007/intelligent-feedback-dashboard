# Intelligent Feedback Dashboard

This project presents an intelligent dashboard for student feedback analysis using Natural Language Processing (NLP) techniques. It processes unstructured textual feedback to perform sentiment analysis and theme extraction, and visualizes insights through an interactive dashboard to support data-driven decision-making in educational institutions.
